svn::has-pending-modifications

Returns true if a svn checkout folder has pending modifications.

Usage

bool svn::has-pending-modifications(path)

Parameters

Name Type Description
path string The local folder path. If null uses Project.BaseDirectory.

Return Value

true if the local folder has pending modifications, false otherwise.

Examples

  • svn::has-pending-modifications() ==> true
  • svn::has-pending-modifications('E:\Sources') ==> false
