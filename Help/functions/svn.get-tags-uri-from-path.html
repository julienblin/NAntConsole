svn::get-tags-uri-from-path

Returns the corresponding /tags svn uri attached to a checkout local folder (if the local folder is a /trunk or a /branch/x).

Usage

string svn::get-tags-uri-from-path(path)

Parameters

Name Type Description
path string The local folder path. If null uses Project.BaseDirectory.

Return Value

The svn '/tags' uri attached to the local folder.

Examples

  • svn::get-tags-uri-from-path() ==> '.../tags'
  • svn::get-tags-uri-from-path('E:\Sources') ==> '.../tags'
