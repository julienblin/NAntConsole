svn::get-uri-from-path

Returns a svn uri attached to a checkout local folder.

Usage

string svn::get-uri-from-path(path)

Parameters

Name Type Description
path string The local folder path. If null uses Project.BaseDirectory.

Return Value

The svn uri attached to the local folder.

Examples

  • svn::get-uri-from-path() ==> '...'
  • svn::get-uri-from-path('E:\Sources') ==> '...'
