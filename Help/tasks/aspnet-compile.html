NAnt Help Task Reference <aspnet-compile> @NAntConsoleVersion@

<aspnet-compile>

Pre-compile ASP.NET websites.

Uses the sdk utility aspnet_compiler.

Parameters

Attribute Type Description Required
srcDir Directory The directory where the website to compiled resides (usually the root of the website project under Visual Studio. True
targetDir Directory The directory where the precompiled website will be generated. True
updatable Bool True to allow the website to be updatable, false otherwise. The default is false. False
debug Bool True to allow the website to be debugable, false otherwise. The default is false. False
fixedNames Bool True to generate fixed assembly names, false otherwise. The default is false. False

Examples

  • Pre-compile a website using fixed names. 

    <aspnet-compile srcDir="E:\Sources\Website" targetDir="E:\Published" fixedNames="true" />
