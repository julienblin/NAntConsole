NAntContrib Help Task Reference <gac-install> @NAntConsoleVersion@

<gac-install>

Installs assemblies into the Global Assembly Cache (GAC) without using the gacutil SDK tool.

Parameters

Attribute Type Description Required
assembly file Assembly file to install. False
failonerror bool Determines if task failure stops the build, or is just reported. The default is true. False
if bool If true then the task will be executed; otherwise, skipped. The default is true. False
unless bool Opposite of if. If false then the task will be executed; otherwise, skipped. The default is false. False

Nested Elements:

<fileset>

Used to select the files to install.

</fileset>

Examples

  • Installs Shared.dll into the GAC.

    <gac-install assembly="Shared.dll" />

  • Installs a group of assembly into the GAC.

    <gac-install>
    <fileset baseDir=".">
        <include name="*.dll" />
    </fileset>
</gac-install>
