NAntContrib Help Task Reference <gac-uninstall> @NAntConsoleVersion@

<gac-uninstall>

Uninstalls assemblies into the Global Assembly Cache (GAC) without using the gacutil SDK tool.

Parameters

Attribute Type Description Required
assembly file Assembly file to uninstall. False
failonerror bool Determines if task failure stops the build, or is just reported. The default is true. False
if bool If true then the task will be executed; otherwise, skipped. The default is true. False
unless bool Opposite of if. If false then the task will be executed; otherwise, skipped. The default is false. False

Nested Elements:

<fileset>

Used to select the files to uninstall.

</fileset>

Examples

  • Uninstalls Shared.dll into the GAC.

    <gac-uninstall assembly="Shared.dll" />
