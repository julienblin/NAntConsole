NAnt Help Task Reference <remove-access-rule> @NAntConsoleVersion@

<remove-access-rule>

Removes access rules (authorization) to files and directories.

Removes all the access rules associated with a ntaccount.

Parameters

Attribute Type Description Required
dir Directory Directory to apply changes to. False
file File File to apply changes to. False
share String The network share to apply changes to. (only FullControl is working). False
ntaccount String NTAccount (DOMAIN\USERNAME) on which the rights are deleted. True

Nested Elements:

<fileset>

Files to apply changes to.

</fileset>

Remarks

Access rules can be removed even if they don't exist. No error will be thrown.

Examples

  • Remove all access rules to user "DOMAIN\User1" on directory "E:\MyApp\Logs". 

    <remove-access-rule dir="E:\MyApp\Logs" ntaccount="DOMAIN\User1" />

  • Remove all access rules to user "DOMAIN\User2" on the files of directory "E:\MyApp\Logs". 

    <remove-access-rule ntaccount="DOMAIN\User2">
    <fileset basedir="E:\MyApp\Logs">
        <include name="*" />
    </fileset>
</remove-access-rule>

  • Remove all access rules to user "DOMAIN\User1" on share "\\SERVER\Share". 

    <remove-access-rule share="\\SERVER\Share" ntaccount="DOMAIN\User1" />
