<replace-com-dll>

Replace a COM dll (carefully unregister/register while taking care of blocking processes).

Parameters

Attribute Type Description Required
source file Source file (the file to replace). True
dest file Destination file (the new file). True

Actions

The following actions are performed :

  • if the dest file exists :
    • kill-process on the dest dll.
    • unregister on the dest dll.
  • copy of the source dll to the dest.
  • com-register on the "new"dest dll.

Examples

  • Replace old.dll by new.dll. 

    <replace-com-dll source="new.dll" dest="old.dll" />
