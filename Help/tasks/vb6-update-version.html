NAnt Help Task Reference <vb6-update-version> @NAntConsoleVersion@

<vb6-update-version>

Updates a .vbp file generation version.

Parameters

Attribute Type Description Required
project file .vbp file to modify. True
version Version Version to set (using Major.Minor.Build). The default takes the value of the version property. False

Examples

  • Sets the version value of "MyProject.vbp" to 1.2.3. 

    <vb6-update-version project="MyProject.vbp" version="1.2.3.4" />
