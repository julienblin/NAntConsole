NAnt Help Task Reference <task-template> @NAntConsoleVersion@

<task-template>

Creates an ordered list of elements.

Parameters

Attribute Type Description Required
list string The name of the list to create. True

Nested Elements:

<item>

Represents an item of the list.

</item>

Examples

  • Creates a list named dll.components and add 2 elements. 

    <list list="dll.components">
    <item value="myproject.dll" />
    <item value="myproject2.dll" />
</list>
