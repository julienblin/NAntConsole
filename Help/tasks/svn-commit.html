<svn-commit>

Commit a local folder under source control to a SVN server.

The TortoiseSVN default credentials will be used.

Parameters

Attribute Type Description Required
dir Directory The local directory to commit. True
message string The log message. True

Examples

  • Commits the local directory "E:\Sources". 

    <svn-commit dir="E:\Sources" message="Committing sources." />
