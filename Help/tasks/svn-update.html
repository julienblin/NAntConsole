NAnt Help Task Reference <svn-update> @NAntConsoleVersion@

<svn-update>

Update a local folder under source control from its SVN server.

The TortoiseSVN default credentials will be used.

Parameters

Attribute Type Description Required
dir Directory The local directory to commit. True

Examples

  • Update the local directory "E:\Sources". 

    <svn-update dir="E:\Sources" />
