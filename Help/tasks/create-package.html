NAnt Help Task Reference <create-package> @NAntConsoleVersion@

<create-package>

Creates a ".deploy" package.

Parameters

Attribute Type Description Required
dir directory Base directory which constitue the root of the package. True
package file Name of the package. True

Content of the package

A ".deploy" package is a zip file constituted by the content of the Dir folder.

The package also includes automatically (which means you don't need to copy these elements in the Dir folder :

  • A copy of the Environment directory if it exists;
  • A copy of the current ".nant" file named deploy.nant;
  • A file named NAntConsole.version which contains the current NAntConsole version number;
  • A file named version.txt which contains the value of the version property if defined. Can be used by the version task.

Examples

  • Creates a package named MyPackage.deploy containing the D:\Package folder and all its sub-folders. 

    <create-package dir="D:\Package" package="MyPackage.deploy" />
