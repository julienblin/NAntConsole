<svn-checkout>

Checkout a url from a SVN server to a local folder.

The TortoiseSVN default credentials will be used.

Parameters

Attribute Type Description Required
dir Directory The local directory to commit. True
url Uri The SVN url. True

Examples

  • Checkout http://svn.cgiquebec.ca/repos/CDS the local directory "E:\Sources". 

    <svn-checkout url="http://svn.cgiquebec.ca/repos/CDS" dir="E:\Sources" />
