<iis-delete-website>

Deletes a IIS website.

Parameters

Attribute Type Description Required
website string The name of the website where the virtual directory is. Use "Default Web Site" for the default web site. True
delete-apppool bool True to delete the associated application pool, false otherwise. The default is true. False

Examples

  • Deletes the website "ReferenceWebSite" and its associated application pool. 

    <iis-delete-website name="ReferenceWebSite" />

  • Deletes the website "ReferenceWebSite" but do not delete the associated application pool. 

    <iis-delete-website name="ReferenceWebSite" delete-apppool="false" />
