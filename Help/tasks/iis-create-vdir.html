NAnt Help Task Reference <iis-create-vdir> @NAntConsoleVersion@

<iis-create-vdir>

Creates a IIS Virtual directory.

Parameters

Attribute Type Description Required
website string The name of the website where the virtual directory will be created. Use "Default Web Site" for the default web site. True
name string The name of the virtual directory to create. True

Nested Elements:

<property>

Set properties applied to the virtual directory in the IIS Metabase.

</property>

<vdir>

Defines a virtual directory inside the current virtual directory. Note that in order to have a usable website, you must create a "root" vdir (accordingly named root).

</vdir>

Wizard

To assist you in creating and configuring a IIS website, you can :
  • Create and configure a IIS virtual directory using the Windows IIS Manager.
  • Launch the "Extract IIS config..." tool found in NAntConsole.
  • Connect to the computer where you set the IIS virtual directory and select the virtual directory.
  • The installation code can then be copied/pasted into the nant script.
