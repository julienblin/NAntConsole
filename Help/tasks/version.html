NAnt Help Task Reference <version> @NAntConsoleVersion@

<version>

Generates a unique version number based on DateTime.Now.

The algorithm is [Year].[Month].[Day].[Number of seconds since midnight].

Important : if a file named .\version.txt exists, the version number will be loaded from its first line.

Parameters

Attribute Type Description Required
property String Name of the property where to store the version number. The default is version. False
failonerror bool Determines if task failure stops the build, or is just reported. The default is true. False
if bool If true then the task will be executed; otherwise, skipped. The default is true. False
unless bool Opposite of if. If false then the task will be executed; otherwise, skipped. The default is false. False

See also

create-package task which writes a version.txt file to seal the version number when deploying.
