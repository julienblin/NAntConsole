NAnt Help Task Reference <at-add-job> @NAntConsoleVersion@

<at-add-job>

Adds a scheduled task.

Parameters

Attribute Type Description Required
name string The name of the scheduled task. A ".job" extension will be added if not present. True
comment string Comment relative to the scheduled task. True
program File The executable to run. True
args string The arguments to pass to the executable. False
workingDirectory string The working directory on which the program will be executed. False

Nested Elements:

<credentials>

Account that will be used to run the program.

</credentials>

<triggers>

Triggers who define when the program will be run.

</triggers>

Examples

  • Creates a scheduled task named "MyScheduledTask" running "C:\myprogram.exe" every day at 5:00 PM . 

    <at-add-job name="MyScheduledTask" comment="My scheduled task" program="C:\myprogram.exe" > 
    <triggers>
        <daily startHour="17" startMinute="0" />
    </triggers>
</at-add-job>

  • Creates a scheduled task named "MyScheduledTask2" running "C:\myprogram2.exe" every monday at 5:00 PM and at every system start using the credentials from DOMAIN\User1. 

    <at-add-job name="MyScheduledTask2" comment="My second scheduled task" program="C:\myprogram2.exe" >
    <credentials domain="DOMAIN" username="User1" password="User1Password" />
    <triggers>
        <weekly weekDays="Monday" startHour="17" startMinute="0" />
        <systemstart />
    </triggers>
</at-add-job>
