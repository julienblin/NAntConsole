NAnt Help Task Reference <iis-create-website> @NAntConsoleVersion@

<iis-create-website>

Creates a IIS website.

Parameters

Attribute Type Description Required
website string The name of the website to create. True

Nested Elements:

<apppool>

Each website is attached to an apppool named [WebsiteName]AppPool which is deleted / created on task execution.
This node allows you to set properties on the created apppool.

</apppool>

<property>

Set properties applied to the Website in the IIS Metabase.

</property>

<vdir>

Defines a virtual directory inside the current website. Note that in order to have a usable website, you must create a "root" vdir (accordingly named root).

</vdir>

Wizard

To assist you in creating and configuring a IIS website, you can :
  • Create and configure a IIS website using the Windows IIS Manager.
  • Launch the "Extract IIS config..." tool found in NAntConsole.
  • Connect to the computer where you set the IIS website and select the website.
  • The installation code can then be copied/pasted into the nant script.
