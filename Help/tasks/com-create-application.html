NAnt Help Task Reference <com-create-application> @NAntConsoleVersion@

<com-create-application>

Creates a COM+ application.

Parameters

Attribute Type Description Required
name string The name of the COM+ application. True

Nested Elements:

<property>

Sets a property on the application.
The list of possible properties can be found here.

</property>

<role>

Role associated with the application.

</role>

<install-component>

Install a VB/COM or .Net component inside application.

</install-component>

<component-property>

Set a specific property of a component (previously installed by install-component).

</component-property>

Wizard

To assist you in creating and configuring a COM+ application, you can :
  • Create and configure a COM+ application using the Windows Component Services panel
  • Launch the "Extract COM+ config..." tool found in NAntConsole.
  • Connect to the computer where you set the COM+ application and select the component.
  • The installation code can then be copied/pasted into the nant script.

Examples

  • Creates a application named SampleServer and sets some roles, properties. Also install a component and set some property. 

    <com-create-application name="SampleServer">
	<property name="AccessChecksLevel" value="COMAdminAccessChecksApplicationLevel" />
	<property name="Identity" value="TestUser" />
	<role name="CreatorOwner">
		<user name="VM-CDS-W2K3-BAI\Administrator" />
	</role>
	<install-component file="ArtistXMLDataLib.dll" />
	<component-property component-name="ArtistXMLDataLib.CDisciplinePersist" property-name="ComponentTransactionTimeout" value="1800" />
</com-create-application>
