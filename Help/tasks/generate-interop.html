NAnt Help Task Reference <generate-interop> @NAntConsoleVersion@

<generate-interop>

Generates a .Net Interop assembly used to reference COM dlls.

Parameters

Attribute Type Description Required
comdll File The COM dll which will be used to generate the .Net interop. True
outdir Directory The output directory where the generated files will be placed. True
force Bool true to force the generation even if the source comdll hasn't changed, false otherwise. The default is false. False

Conventions

The following actions will be taken :

  1. Unless force="true", the comdll will be binary compared with a file named [outdir]\Interop.[comdllfilename].reference if it exists. If the files are equal, nothing will be done.
  2. If a file named [outdir]\Interop.[comdllfilename].snk doesn't exist, it will be generated by the sn Sdk tool.
  3. The generated interop will be [outdir]\Interop.[comdllfilename].dll with a namespace of [comdllfilename].
  4. The comdll will be copied to [outdir]\Interop.[comdllfilename].reference for future comparison (see 1.).

Examples

  • The following will create 3 files :

    • E:\Dependencies\Interop\Interop.MyComponent.reference
    • E:\Dependencies\Interop\Interop.MyComponent.snk
    • E:\Dependencies\Interop\Interop.MyComponent.dll, with a namespace of MyComponent

    <generate-interop comdll="E:\Dependencies\Com\MyComponent.dll" outdir="E:\Dependencies\Interop" />
