<svn-copy>

Remote copy a local folder under source control to another location on the SVN server. The remote repository will be determined by the local folder "SVN attachment".

The TortoiseSVN default credentials will be used.

Parameters

Attribute Type Description Required
dir Directory The local directory to commit. True
uri uri The svn uri to copy the source repository tree. The final directory must not exists prior to the copy. True
message string The log message. True

Examples

  • Copy the local directory "E:\Sources" to the repository "http://svn/MyApp/tags/1.2.3.4". 

    <svn-copy dir="E:\Sources" uri="http://svn/MyApp/tags/1.2.3.4" message="Tagging release 1.2.3.4." />
