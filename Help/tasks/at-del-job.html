NAnt Help Task Reference <at-del-job> @NAntConsoleVersion@

<at-del-job>

Deletes a scheduled task.

Parameters

Attribute Type Description Required
name string The name of the scheduled task. A ".job" extension will be added if not present. True

Examples

  • Deletes a scheduled task named "MyScheduledTask". 

    <at-del-job name="MyScheduledTask" />
