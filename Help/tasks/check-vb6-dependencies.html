<check-vb6-dependencies>

Checks dll dependencies listed in .vbp VB6 project files.

Parameters

Attribute Type Description Required
vb-project file Specify the name of a .vbp file. True

Rules

  • each reference inside vb-project file is checked and fail :
    • if the path is rooted instead of relative (e.g. C:\Myproject\Myproject.dll fail whereas ..\..\Myproject.dll doesn't);
    • if the path reference a non-existent file (the file pointed by the path reference must exists);
    • if the classid and the version in the reference are different than those of the reference file pointed by the path.
  • if the path reference the system directory (e.g. C:\WINDOWS\System32) or the Program files (e.g. C:\Program Files), the check is ignored ;

Examples

  • Checks a .vbp files dependencies. 

    <check-vb6-dependencies vb-project="CDSUtils\CDSUtils.vbp" />
