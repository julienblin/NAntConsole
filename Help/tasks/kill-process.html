NAnt Help Task Reference <kill-process> @NAntConsoleVersion@

<kill-process>

Kill a process by its name or kill all the processes blocking a dll.

Parameters

Attribute Type Description Required
dll File The name of the dll used to kill processes that blocks it. False
process-name string The name of the process to kill. False

Examples

  • Kill the process "NAntConsole.exe" 

    <kill-process process-name="NAntConsole.exe" />

  • Kill all the processes blocking "D:\Components\MyConponent.dll" 

    <kill-process dll="D:\Components\MyConponent.dll" />
