<env-include>

Include files depending on the machine where the execution takes place.

Rules

<env-include /> will try to include files in the .\Environment folder following these rules :

  • the file names must be *.*.config (e.g. lsa.dev.config);
  • all the files named *.[machinename].config;
  • if an environment variable named ENV has been set, all the files named *.[ENV-value].config if * as not been included previously;
  • all the files named *.default.config if * as not been included previously.

Examples

  • Call env-include. 

    <env-include />
