NAnt Help Task Reference <write-registry> @NAntConsoleVersion@

<write-registry>

Writes registry values.

Parameters

Attribute Type Description Required
value string Value to be stored in the registry. True
key string The registry key to delete, including the path (e.g. SOFTWARE\NAntConsole). If the last component of the path is *, the whole subkey is removed. True
hive string Space separated list of registry hives to search for. For a list of possible values see RegistryHive. The default is LocalMachine. False

Examples

  • Writes the value "2" in HKEY_LOCAL_MACHINE\Software\NAntConsole\Version. 

    <write-registry value="2" key="Software\NAntConsole\Version" />

  • Writes the value "C:\Programs\NAntConsole" in HKEY_LOCAL_MACHINE\Software\NAntConsole\InstallDir and HKEY_CURRENT_USER\Software\NAntConsole\InstallDir. 

    <write-registry value="C:\Programs\NAntConsole" key="Software\NAntConsole\InstallDir" hive="LocalMachine CurrentUser" />
