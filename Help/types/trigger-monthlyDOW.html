NAnt Help Type Reference <monthlyDOW> @NAntConsoleVersion@

<monthlyDOW>

Trigger that fires on specific days, weeks and months.

Parameters

Attribute Type Description Required
beginDate DateTime The beginning date of the trigger. The default is today. False
endDate DateTime The end date of the trigger. The default is unbounded. False
disabled bool True to disable the trigger, false otherwise. The default is false. False
durationMinutes int Number of minutes after the trigger fires that it remains active. False
intervalMinutes int Number of minutes between executions for a task that is to be run repeatedly. False
killAtDurationEnd bool true to kill (terminate) the task when DurationMinutes expires, false otherwise. The default is false. False
startHour short Hour of the day that trigger will fire (24 hour clock). True
startMinute short Minute of the hour that trigger will fire. True
months MonthsOfTheYear Day of the week on which the trigger fires. False
weekDays DaysOfTheWeek Day of the week on which the trigger fires. True
whichWeeks WhichWeek Weeks of the month in which trigger will fire. True

Examples

  • Creates a trigger that fires every second monday each month at 11:00 AM. 

    <monthlyDOW weekDays="Monday" whichWeeks="SecondWeek" startHour="11" startMinute="0" />
