NAnt Help Type Reference <triggers> @NAntConsoleVersion@

<triggers>

Represent a list of triggers used by the at-add-job task.

Nested Elements:

<daily>

Trigger that fires at daily intervals.

</daily>

<weekly>

Trigger that fires at weekly intervals.

</weekly>

<monthly>

Trigger that fires at monthly intervals.

</monthly>

<monthlyDOW>

Trigger that fires on specific days, weeks and months.

</monthlyDOW>

<once>

Trigger that fires at a particular time, but only once.

</once>

<idle>

Trigger that fires when the system is idle.

</idle>

<logon>

Trigger that fires when a user logs on.

</logon>

<systemstart>

Trigger that fires when the system boots up.

</systemstart>
