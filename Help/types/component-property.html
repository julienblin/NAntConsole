NAnt Help Type Reference <component-property> @NAntConsoleVersion@

<component-property>

Set a specific property of a component (previously installed by install-component).

Parameters

Attribute Type Description Required
component-name string Name of the component to set the property on. True
property-name string Name of the property to set. True
value string Value of the property. True

Property reference

The list of the possible properties can be found here.

Examples

  • Set the timeout of the component ArtistXMLDataLib.CDisciplinePersist to 1800. 

    <component-property component-name="ArtistXMLDataLib.CDisciplinePersist" property-name="ComponentTransactionTimeout" value="1800" />
