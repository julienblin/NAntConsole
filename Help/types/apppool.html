NAnt Help Type Reference <apppool> @NAntConsoleVersion@

<item>

Defines an IIS application pool (used by the <iis-create-website> task).

Nested Elements:

<property>

Set properties applied to the AppPool in the IIS Metabase.

</property>

Examples

  • Set the property AppPoolState on the apppool. 

    <apppool>
	<property name="AppPoolState" value="2" type="DWORD" />
</apppool>
