NAnt Help Type Reference <vdir> @NAntConsoleVersion@

<vdir>

Represent a nested virtual directory on the IIS Metabase.

Parameters

Attribute Type Description Required
name string The name of the virtual directory. True

Nested Elements:

<property>

Set properties applied to this virtual directory in the IIS Metabase.

</property>

<vdir>

Defines a virtual directory inside the current virtual directory.

</vdir>

Examples

  • Defines a virtual directory named "root" and apply a property. Sets also two sub virtual directories. 

    <vdir name="root">
	<property name="AppFriendlyName" value="ReferenceWebSite" type="String" />
	<vdir name="firstChildren">
		<property name="AppFriendlyName" value="Children" type="String" />
	</vdir>
	<vdir name="secondChildren">
		<property name="Path" value="E:\" type="String" />
	</vdir>
</vdir>
