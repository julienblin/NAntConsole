NAnt Help Type Reference <iis-typed-property> @NAntConsoleVersion@

<iis-typed-property>

Represent a property on the IIS Metabase by a name, a value and a type.

Parameters

Attribute Type Description Required
name string The name of the property (as represented in the IIS Metabase). True
value string The value of the property when the type is defined as String or DWORD - null otherwise. False
type string Type can be String, DWORD or MultiString - The default is String. False

Nested Elements:

<entry>

Defines a value when the type is MultiString.

</entry>

Examples

  • Defines a property named AppFriendlyName with a value "ReferenceWebSite" and a String type. 

    <property name="AppFriendlyName" value="ReferenceWebSite" />

  • Defines a property named ServerBindings with two values and a MultiString type. 

    <property name="ServerBindings" type="MultiString" >
    <entry value=":80:www.ref.website.net" />
    <entry value=":443:www.ref.website.net" />
<property>
