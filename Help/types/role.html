NAnt Help Type Reference <role> @NAntConsoleVersion@

<role>

Represent a role associated with a com-create-application task.

Parameters

Attribute Type Description Required
name string Name of the role. True

Nested Elements:

<user>

Represent a user associated with a role.

</user>

Examples

  • Creates the role Customer an associates with two users. 

    <role name="Customer">
    <user name="DOMAINNAME\Administrator" />
    <user name="DOMAINNAME\User2" />
</role>
