NAnt Help Type Reference <install-component> @NAntConsoleVersion@

<install-component>

Install a VB/COM or .Net component inside com-create-application task.

Parameters

Attribute Type Description Required
file file Component (.dll) to install. True
net bool true to indicate a .net assembly, false otherwise. Default is True. False

Nested Elements:

<fileset>

Can be used to specify a range of files to install.

</fileset>

Examples

  • Install a VB/COM component. 

    <install-component file="MyComponent.dll"/>

  • Install a .Net component. 

    <install-component file="MyDotNetComponent.dll" net="true" />

  • Install VB/COM components. 

    <install-component>
	<fileset basedir="Bin">
		<include name="*.dll" />
	</fileset>
</install-component>
